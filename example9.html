Example: Multiple Apps in one Page

The beauty of RobJS is that you can easily plug it in anywhere - and multiple times - in your page.

You might create your app with some backend template engine which gives you a lot of advantages (like Session Authentication, shareable URLs and SEO) but occasionally you need more fluent and interactive parts. RobJS is perfect for that - with literally no setup.

To see how this can be done check out the source code of this website as all provided examples are done with RobJS - intergrated into a normal html-Page.
